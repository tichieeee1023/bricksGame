# 도전! 아케이드 벽돌깨기 게임 (Retro Brick Breaker)

Canvas API와 순수 JavaScript(Vanilla JS)를 활용하여 구현한 90년대 오락실 스타일의 레트로 벽돌깨기 게임입니다. 
웹디자인개발기능사 및 프론트엔드 역량 강화를 위해 기존 기본 예제에 다채로운 네온 디자인 요소와 유저 편의 기능, 화려한 올클리어 이펙트를 커스텀하여 추가했습니다.

---

## 추가 및 개선된 핵심 요소

1. 사이버펑크 / 레트로 오락실 테마 디자인
   - 기존의 단조로운 색상에서 탈피하여 네온 핑크, 시안 블루, 골드 등의 트렌디한 컬러셋을 적용했습니다.
   - 캔버스 전반에 `shadowBlur`와 그림자 효과를 주어 빛나는 브라운관 전광판 효과를 연출했습니다.

2. 동적 난이도 모드 (EASY / HARD)
   - 게임 시작 전 유저가 난이도를 선택할 수 있는 인트로 스크린 팝업을 구현했습니다.
   - 선택한 난이도에 따라 공의 속도, 패들의 크기뿐만 아니라 전체 캔버스의 배경 그라디언트와 테두리 조명 색상까지 실시간으로 동적 변경됩니다.

3. 하단 경고 시스템 (유저 가이드)
   - 키보드 조작과 마우스 조작이 동시에 일어날 때 캔버스 좌표 조작이 꼬이는 현상을 방지하기 위해 화면 하단에 깜빡이는 오락실 전광판 스타일의 조작 가이드 안내문(Warning)을 컴포넌트로 추가했습니다.

4. 영롱한 RGB 무지개 올클리어 엔딩 크레딧
   - 게임 승리(All Clear) 시 광과민성을 유발하는 자극적인 깜빡임 대신, 눈이 편안하면서도 화려한 CSS Gradient 롤링 기법을 도입했습니다.
   - 테두리(Rim Light)뿐만 아니라 캔버스 중앙의 메인 텍스트인 `ALL CLEAR!` 문구 내부까지 무지개 빛 스펙트럼이 부드럽게 흐르는(Flowing) 고급 텍스트 그라디언트 애니메이션을 구현하여 연출력을 극대화했습니다.

## 조작 방법 및 안내사항

* 기본 조작 방법 (아래 방식 중 하나를 선택하여 플레이하세요)
  * MOUSE: 캔버스 안에서 마우스를 좌우로 움직이면 바(Bar)가 마우스 포인터 중심을 따라 부드럽게 이동합니다.
  * KEYBOARD: 키보드 좌우 방향키(`ArrowLeft` / `ArrowRight`) 또는 `A` / `D` 키를 이용하여 바를 조작할 수 있습니다.
* ⚠️ WARNING (필독)
  * 마우스 조작과 키보드 조작을 혼용할 경우 패들의 위치 계산이 충돌할 수 있으니, 반드시 한 가지 조작 모드만 선택하여 플레이하는 것을 권장합니다.

## 테스트용 개발자 치트키

벽돌 35개를 모두 깨지 않고도 올클리어 화면 및 최종 연출(무지개 테두리 및 흐르는 RGB 텍스트 이펙트)을 칼같이 검증하실 수 있도록 개발자 전용 비밀 단축키(디버깅 모드)를 심어두었습니다.
